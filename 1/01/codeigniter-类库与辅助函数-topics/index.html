单元测试类(自带的 Unit Test)

/libraries/unit_testing.html

注意CI 的 UnitTest 只是简化的测试模块，如果需要更强大的 还是建议 PHP Unit (但是问题是无法共享 session 等)

使用

Class Unittest extends Controller {

    function Unittest()
    {
        parent::Controller();
        $this->load->library('unit_test'); 
    }
      
    function index()
    {
        $this->load->model('test_model');

        $this->unit->run($this->test_model->func_name('22'), 'is_array', 'testname');

        // where $this->test_model->func_name('22') is the test data ;
        // is_array : expected datatype
        // testname : name to your test

        echo $this->unit->report();
    }
}

function test_func() {
    $this->load->library('unit_test');

    //格式
    //$this->unit->run(‘test’, ‘expected result’, ‘test name’, ‘notes’);

    //例如
    $test = 1 + 1;
    $expected_result = 2;
    $test_name = 'Adds one plus one';
    $this->unit->run($test, $expected_result, $test_name);
}

第二个参数 expected result期望值，可以是数据，也可以是“类型”

$this->unit->run('Foo', 'Foo');

$this->unit->run('Foo', 'is_string');

有这些类型判断

  • is_object
  • is_string
  • is_bool
  • is_true
  • is_false
  • is_int
  • is_numeric
  • is_float
  • is_double
  • is_array
  • is_null
  • is_resource

生成报告

可以对单独一个测试来输出报告， 或者在全部测试跑完后，一次性输出

echo $this->unit->run($test, $expected_result);

// html 格式
echo $this->unit->report();

// 原始格式 raw
echo $this->unit->result();

严格模式

默认情况下是松散模式

// 1 也可以和一个 布尔值 相等
$this->unit->run(1, TRUE);

启用严格模式，就会判断类型了

$this->unit->use_strict(TRUE);

disable: 保留单元测试代码，但是不运行

$this->unit->active(FALSE);

显示自定义内容

在报告中会显示这些内容

Test Name (test_name) Test Datatype (test_datatype) Expected Datatype (res_datatype) Result (result) File Name (file) Line Number (line) Any notes you entered for the test (notes)

如果需要限定某些 item

$this->unit->set_test_items(array('test_name', 'result'));

或者新建一个模板

$str = '
<table border="0" cellpadding="4" cellspacing="1">
{rows}
        <tr>
                <td>{item}</td>
                <td>{result}</td>
        </tr>
{/rows}
</table>';

// 必须先定义模板
$this->unit->set_template($str);
