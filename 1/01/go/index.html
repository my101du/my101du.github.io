下载 安装

下载二进制包安装后，需要设置环境变量（默认安装到 /usr/local/go 目录）

修改 .bash_profle

export GOROOT=/usr/local/go/bin/
export PATH=$GOROOT:$PATH

然后使之生效source ~/.bash_profile

再添加 GOPATH(通过 go get 安装的第三方工具),假设都下载到 ~/go 目录

export GOPATH=~/go
export PATH=$GOROOT:$GOPATH/bin:$PATH

下载第三方包 go get

例如

go get -u github.com/rakyll/hey

在当前目录得到一个 go/src/… 结构的子目录 存储了所有下载的包的源文件

另外一个 go/bin目录则是可执行文件

运行命令

go/bin/hey -n 500 -c 100 https://www.sample.org

以后你可以把这个用户目录下的 go 目录设置成环境变量 GOPATH, 添加到系统环境变量列表里

vi .bash_profile
export GOPATH=$HOME/go
export $GOPATH/bin:$PATH

这样以后安装的第三方命令就可以直接运行了。（就像 npm -g 安装的工具一样）

最终

参看“科学上网”，使用 privoxy 并设置 git 和 ssh 同时跑代理

vi .bash_profile
# 添加下面内容
export http_proxy=http://127.0.0.1:8118
export http_proxy=https://127.0.0.1:8118

# 生效
source .bash_profile

# 修改 git 配置
git config --global http.proxy http://127.0.0.1:8118
git config --global https.proxy http://127.0.0.1:8118

# 启动 privoxy
privoxy /etc/privoxy/config

取消git 代理

git config --global --unset http.proxy
git config --global --unset https.proxy
