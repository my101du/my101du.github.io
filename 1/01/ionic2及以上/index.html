知识学习顺序

原来的 ionic Javascript ---> AngularJS -----> Ionic

现在要多学一个 TypeScript了 + 新的 AngularJS，而且现在的 AngularJS 和最初版基本可以说变化巨大了。 Javascript ----> TypeScript -----> AngularJS 2 -----> Ionic2

安装

# 安装环境、cli 工具
npm -g install ionic cordova

# 创建一个 turtorial 模板的 ionic 项目
ionic start test-project turtorial

# 如果想要创建基于 ionic1 旧版本的项目
ionic start cutePuppyPics -type ionic1

# 或者项目并不会使用到设备的硬件功能，可以让项目不包含 cordova 库
ionic start test-desktop sidemenu --no-cordova

在浏览器中运行

$ cd cutePuppyPics
$ ionic serve

为了发布到 iOS 和 Android 平台要准备的

iOS 和 Android 平台都有一些自己的独特性，除了 Ionic 和 Codova 你还要了解这些差异性以及如何解决（例如 iOS 需要对 Xcode 的配置、开发者账号配置、iOS 应用的签名等；Android 编译则需要安装和配置 SDK 的版本、运行路径、模拟器）

https://cordova.apache.org/docs/en/latest/guide/platforms/ios/ https://cordova.apache.org/docs/en/latest/guide/platforms/android/

学习 TypeScript （以及前端开发的一系列知识）

http://ionicframework.com/docs/developer-resources/what-is/#typescript

还有ES5 ES6 webpack babel 等等

从老版本中迁移到新版

Angular 1 语法变化

Ionic Native

通常运行在 webview 内的 html 和 js 是无法调用系统的硬件的（例如摄像头 GPS等）， 但是 Cordova 可以让 Hybrid Application 使用原生特性来调用系统的硬件。

老版本的 Ionic 使用的是 Javascript，因此可以直接使用 Cordova 的一些 plugins。但是在新版的 Ionic 中使用 TypeScript 语言，因此需要再加一个“Wrapper”来调用 Cordova 及相关插件，这就是Ionic Native

Promise 和 Observables

Ionic Native 把插件的 callbacks 封装在一个 Promise 或者一个 Observable，为所有插件提供了一个通用的接口，确保在 Angular 中的原生事件触发器检查。

import { Geolocation } from '@ionic-native/geolocation';
import { Platform } from 'ionic-angular';

class MyComponentOrService {

  constructor(private platform: Platform, private geolocation: Geolocation) {
    platform.ready().then(() => {
      // get current position
      geolocation.getCurrentPosition().then(pos => {
        console.log('lat: ' + pos.coords.latitude + ', lon: ' + pos.coords.longitude);
      });

      const watch = geolocation.watchPosition().subscribe(pos => {
        console.log('lat: ' + pos.coords.latitude + ', lon: ' + pos.coords.longitude);
      });

      // to stop watching
      watch.unsubscribe();
    });
  }
}

Runtime Diagnostics

Ionic Native 能够检查插件是否安装或是否有问题。。。（怎么做？）

安装 ionic-native core package

npm install @ionic-native/core --save

安装插件

第一步是为每一个插件安装 Ionic Native package

npm install @ionic-native/camera --save

第二步才是安装真正的 Cordova 插件

ionic cordova plugin add cordova-plugin-camera

插件包安装完后要添加到 NgModule

import { Camera } from '@ionic-native/camera';

@NgModule({
   providers: [
    Camera
  ]
})
export class AppModule { }
