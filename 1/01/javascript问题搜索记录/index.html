var key = "happyCount";
var obj = {};
obj[key] = someValueArray;

中处理 checkbox 的选中/清除选中状态，以及 attr() 与 prop()

首先，在 jQuery 1.6 之前，使用 attr() 可能会返回一些未预料的值。 在这个版本之后，建议使用prop()替代

处理 checkbox 的选中，这里有个藏着的问题

如果用下面的代码让 checkbox 选中

$('input[type=checkbox]').attr({"checked": "checked"});

再使用下面任意一个方法取消选中

$('input[type=checkbox]').attr({"checked": false});
$('input[type=checkbox]').removeAttr("checked");

再次用第一步的.attr({"checked": "checked"})去选中，会发现无效了！checkbox 仍然一直处于取消选中。（这在重复利用一个编辑表单加载不同 ajax 返回数据的时候会遇到）

// 即使设置了选中，在表单上也不会生效
$('input[type=checkbox]').attr({"checked": "checked"});

要解决这个问题，必须成对匹配使用 prop()，并分别设置true和false来实现

$('input[type=checkbox]')..prop({"checked":true});
$('input[type=checkbox]')..prop({"checked":false});

//再次选中 ，表单上的 checkbox 有效了
$('input[type=checkbox]')..prop({"checked":true});

substr() 与 substring() 截取字符串的不同

substr(start, [length]) 返回截取从开始位置，并指定长度的字符串（共有length位） substring(start, [end])返回截取指定开始/结束位置之间的字符串（从start位到end位之间）

例如

const str = 'abcdefg';

str_1 = str.substr(1, 3);  //bcd
str_2 = str.substring(1, 3);  //bc
