MarketPlace

购买云服务

搭建在自己的服务器上

10美元可以用在10人以下，试用30天

mac 上安装

下载 tar.gz

# 确保安装了 java
java -version

设置 JIRA_HOME .

cd ~/download/jira/folder/bin
sh config.sh

指向这个目录

假设这里用 Postgre SQL (因为我本地装的是 MySQL 5.5….)

先把数据库建好

然后填写信息并测试下链接

然后为 JIRA 创建一个专用账号，（避免和其他账号干扰）例如

linux

$ sudo /usr/sbin/useradd --create-home --comment "Account for running JIRA" --shell /bin/bash jira

windows

> net user jira mypassword /add /comment:"Account for running JIRA"

确保下面的目录可以被这个账号写入非常重要

logs temp work JIRA Home Directory

如果没有给写入权限，后面会报错 import/startupdatabase.xml 找不到

Caused by: java.nio.file.NoSuchFileException: /Users/ericzhang/Downloads/atlassian-jira-software-8.2.1-standalone/atlassian-jira/import/startupdatabase.xml

切换到新创建的专用账号。

（如果是本地测试， 用 sudo sh start-jira.sh 启动， 这样就表示当前登录的管理员账号有权限修改文件了 加sudo）

启动

sh bin/startup.sh

（Tomcat 作为 web 服务器被启动，和在 config.sh 里的是一样的），

访问 http://localhost:8080.

如果权限正确、目录（JIRA_HOME）正确、数据库配置正确

初始化 jira

可选插件

  • Portfolio for JIRA

agile roadmapping tool, build a plan, see the big picture, track progress, and easily share with stakeholders (规划与检视, 将战略目标与开发联系起来)

  • Jira Service Desk

Solve, prioritize, and automate tickets faster than ever （Ticket的分配和处理，客服人员解决问题）

  • Confluence Organize your work, create documents, and discuss everything in one place （提出问题、获得解答）

  • Bitbucket （源代码管理）
