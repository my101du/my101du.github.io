ajax 提交

https://stackoverflow.com/questions/1960240/jquery-ajax-submit-form

通过 ajax 提交文件上传，组装数据，并禁止自动序列化

jQuery的$.ajax()方法的时候参数processData默认为true(该方法为jQuery独有的) 默认情况下会将发送的数据序列化以适应默认的内容类型 application/x-www-form-urlencoded

如果想发送不想转换的的信息的时候需要手动将其设置为 false

在我遇到的是传输的是blob对象的时候就是不需要将传输的数据序列化,一般的还有类似DOM树等

// 监听文件上传动作
$('#fileupload').change(function (img) {
	if (!$('#fileupload').val()) {
		return;
	}

	// 手动创建一个 FormData 对象
	var postData = new FormData();

	// 填充数据 {file: ...,  age: ...}
	postData.append('file', this.files[0]);
	// 更多，如 postData.append('age', $("age").val());

	$.ajax({
		async: true,
		url: 'post/path/action',
		type: 'POST',
		data: postData,
		dataType: 'json',
		processData: false, // 这里一定要设为 false, 因为上传的是文件。告诉 jQuery 不要序列化它
		contentType: false, // 这里也是 false, 告诉 jQuery 不要设置 content-type
		success: function (data, textStatus, jqXHR)
		{
			$('#fileupload').val('');
			if (data.status == 'OK') {
				messageBoxError('Import Done');
			} else {
				messageBoxError(data.status);
			}

			waitMsgOFF('body');

		},
		error: function (jqXHR, textStatus, errorThrown)
		{
			waitMsgOFF('body');
			$('#fileupload').val('');

			if (checkSessionExpired(jqXHR.responseText)) {
				return;
			}

			//console.log(jqXHR, textStatus, errorThrown);

			toastErrorBig(jqXHR.responseText);
			//$('#docRepProgressBar').hide();
			//unlockMain();

		}
	});
});

事件

为动态创建的元素绑定事件

https://www.youtube.com/watch?v=Wxnd21_f_pc

在动态创建元素的时候，就要给绑定事件，否则不会触发，例如

var button = $('<button class="test">create element</button>');

// create element dynamic
button.click(function(){
    $('.root').append('<span class="new_item">');
})

$('.new_item').click(function(e){
    alert('click');
});

无反应。 改成直接在创建元素完成后绑定事件，再插入到 DOM

var button = $('<button class="test">create element</button>');

// create element dynamic
var new_element = '<span class="new_item">').click(function(e){
        alert('click');
    });

button.click(function(){
    $('.root').append(new_element)
});

break, return 等

** 这个例子没举好** 要改。 有错误。

$.each([1,2,3,4,5,6], function (i, v) {
	if(i == 5) 

		return false;
});

foo();

注意这里有个坑，each 是异步执行的，我们期望的是发现 i=5 就不往后执行了。结果 foo() 还是执行。

为了避免，需要加上计数器

var j = 0;
$.each([1,2,3,4,5,6], function (i, v) {
	j++;
});

if(j == 5){
	foo();
}

可以改成 for 同步的方式来执行
