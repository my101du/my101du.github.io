推送

设置 Amazon 的设备与内容

设备

在移动设备/电脑上安装 kindle 软件并登陆后，会自动把设备添加进来。复制某个设备的邮件地址，等下粘贴到内容推送源

允许的电子邮箱

推送源

看云 https://www.kancloud.cn/setting

使用 curl 批量推送看云内容到 kindle

chrome 分析 network，点击 “推送到kindle” 是一个 post 动作，但是必须要带 cookie 参数，否则会返回错误内容

在这里也可以看到该站点写到浏览器的 sessionid

手机上也自动下载了

写一段代码来批量获取它们。 以官方账号的书籍集合 https://www.kancloud.cn/@kancloud?tab=book&page=1 为例（共有23页）

推荐列表 https://www.kancloud.cn/explore/recommend?page=51

你可以使用自己喜欢的语言来写“采集器”，这里我用了simplehtmldom (http://simplehtmldom.sourceforge.net/，一个可以用 jQuery 类似语法)

可能这个库太老了，在 php7 一直无法 find 到内容，换了这个 https://github.com/paquettg/php-html-parser

require "vendor/autoload.php";
use PHPHtmlParser\Dom;

$dom = new Dom;

for($i=1; $i<=23; $i++){
	$dom->loadFromUrl('https://www.kancloud.cn/@kancloud?tab=book&page=' . $i);

	$links = $dom->find('.item a');
	foreach ($links as $link)
	{
		echo 'curl -b PHPSESSID=xxxxxxxx -X POST https://www.kancloud.cn' . $link->getAttribute('href') . '/kindle' . '<br />';
	}
}

把得到的结果保存成 download.sh 运行

不过有频率限制。。。。注意控制下速度
