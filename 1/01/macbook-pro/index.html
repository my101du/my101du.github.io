外接显示器

2015 之前版本不支持 4k 跑到 60Hz 问题

https://kandrejevs.com/blog/read/50/More-than-30Hz-on-Your-133-Macbook-Pro-and-4K-monitor

能跑到 52Hz

可以用 swtichResX

  1. 添加 Custom resolutions
  2. 设置 Pixel Clock 为 450,00 MHz

彻底关闭自带的显示器

亮度大法 外接后，把笔记本的显示器亮度降到最低。这样还是会有显卡资源的消耗，只是自欺欺人

磁铁大法 把磁铁放在键盘左侧，让机器误以为合盖，把显示器关闭。。欺机

系统命令行法

重启，按 cmd + R 进入维护模式，打开终端

sudo nvram boot-args="niog=1"

严格遵守这个步骤才能实现

  1. 连上电源和外接显示器
  2. 按开机键，立即合盖
  3. 等外接显示器有信号时 打开盖子
  4. 观察是否已经关闭

问题: 休眠唤醒时会点亮mac屏幕 外接键盘 合盖 敲外接键盘或鼠标点亮外接显示器 键盘按command+F2(显示器闪一下) 开盖

如果以后要恢复

sudo nvram -d boot-args

2013~2014 款 mbp 的理论最高速度为 2GB/s,选择 m.2 接口 ssd ,没必要上高端系列，PCI-E 2.0 的接口会限制高端 ssd 的发挥，intel 760p 就挺好用。2015 款 mbp ,接口升级为PCI-E 2.0, 理论最高速度翻倍，达到 4GB/s。在预算充足的条件下，大可随意购入高端 ssd ，以释放全部的效能。

步骤

制作启动盘

在 Mac App Store 中搜索 Mojave 并下载。 磁盘工具格式化U盘 「Mac OS X 扩展（日志式）」格式、GUID 分区图，并将 U 盘命名为「Mojave」 终端 sudo /Applications/Install\ macOS\ Mojave.app/Contents/Resources/createinstallmedia --volume /Volumes/Mojave /Applications/Install\ macOS\ Mojave.app --nointeraction

拆机

10颗螺丝，注意有两类螺丝：2 颗 2.3 毫米和 8 颗 3.0 毫米梅花五角螺丝 拨片等工具从上往下（参看上图）慢慢翘起后盖，这里有个小技巧，翘起一侧后，可以用塑料撬棍撑起一边，然后拆另一侧。 拆开后盖后，第一时间用撬棍从侧面将电池电源切断 清灰工作，先用毛刷，气吹等工具，简单清理一下设备内部。然后用压缩空气除尘罐着重清理出风口，风扇等区域，注意灰尘防护。 拆开拆卸固定SSD的 2.9 毫米 T5 Torx 螺丝，取下原装 ssd，取下时可先将 ssd 向上稍抬起，后直接抽出 将购入第三方 ssd 装上转换接口，插入 ssd 槽中，并固定。 接上电源，先不要安装后盖，下面安装新系统。

安装系统

更换完毕后,重启电脑，开机时按住 option 键不放，直到进入磁盘选择界面，选择你自己的磁盘。

格式新换的 ssd(抹掉,格式为 APFS ，方案为 GUID 分区图) c.格式完之后，退出磁盘工具，选择安装 MacOS 系统到新格式化 ssd,等待安装完成。 登陆iCloud账号后，自动同步数据。上一步也可以选择从 时间机器 里迁移数据到新的 ssd 里面 e.开启 ssd TRIM ,固态硬盘的TRIM能避免SSD在垃圾回收时将无用数据当成有用数据来搬运，从而延长 ssd 寿命。在终端中输入命令行sudo trimforce enable，输入系统密码后，按照提示开启。

装回后盖，检查

确保系统运行一切正常后，安装好后盖，注意不同螺丝的位置。 在 “关于本机 → 系统报告 → NVMExpress” 中,可看到已经识别出 NVME ,同时 TRIM 显示支持，链接速度为 X4 ，一切正常。

还有休眠问题在新版 macOS 里解决了

开机变慢问题 23 楼

hibernatemode 和 standby 的设置 14.3 默认参数 stanby 1 没有睡眠问题

休眠的确是有些问题的，我直接把所谓的 “write to ssd” 大概算是 deep sleep 模式给关掉了，唤醒比以前更加无痛，休眠期间耗电也没有明显提高。

参考命令：sudo pmset -a hibernatemode 0 standby 0 autopoweroff 0 ；

版本

[怀疑乱写的？]这个问题我是这么解决的，原装硬盘系统整到10.3.6，然后插上760p，重装系统到10.3.6 然后通过appstore直接更新到10.14.3 然后就好了

原装硬盘先升级到最新10.13或10.14.4；制作U盘安装盘（教程网上有），拆掉旧SSD，装上新SSD，用U盘重新装10.13.6或10.14就可以了，我也是2014 mid 13，intel760P；用了某迪和另外一家的转接（黑色后面有NVME字样），最后选了另外一家的，某迪的卡得太紧了；休眠模式用系统默认的3，可正常休眠；调到25出现休眠唤醒后开机灰屏。现已将休眠模式改回3。

电池供电，盒盖，模式 3，默认在24小时后也会进入 25 模式的

单来说，如果是2013年末到14年中的MBP，13年末到14年初的MBA，换NVME SSD必出休眠问题 那些说没出的，要么修改了主板bootrom（很麻烦），要么是休眠模式不对，MACBOOK有三种休眠模式，0（相当于不休眠），3（内存供电，硬盘备份），25（真正的休眠，内存不供电，原内存信息储存在硬盘，唤醒时把信息从硬盘还原到内存），会出问题的只有25,因为只有25会真正清空内存中的内容，也只有25不掉电。 如果你把休眠模式切换到25（方法百度），然后再尝试休眠，没有问题，才是真正的没有问题。

解决这个问题两个方法，第一个，禁用休眠，让你的电脑永远处在0模式（也就是一直开着），或者3模式（但是外网论坛给的解决方式是0模式我也不知道为什么） 代码在这： sudo pmset -a hibernatemode 0 standby 0 autopoweroff 0

第二个,修改bootrom，方法在这 forums.macrumors.com/thread…ssd-to-m-2-nvme.2034976/page-65#post-26224320

如果允许的话，希望能众筹一条刷BOOTROM的线，让大家都能真正的解决这个问题。

本人 MacBook Pro 2014 15’ 刷 BOOTROM 之后完美解决睡眠问题。

合盖睡死 问题(集中在2013、2014年份全部 MacBook Pro、MacAir 上)

把macbook的睡眠功能关闭，因为macbook睡眠的时候需要把内存的内容写入硬盘，可能在这个过程中有兼容问题，导致开不了机，关闭就好了，也不怎么耗电。(如果需要恢复设置，到『系统设置 - 能耗』，点『恢复默认』就行。)

sudo pmset hibernatemode 0 standby 0

开机选择硬盘时，新SSD是黄色

Macbook把新硬盘识别为移动硬盘了，暂时没什么好的办法解决，幸运的是并不影响性能，也没有兼容问题，进系统内后仍会把它识别为内置硬盘。2015以后的版本好像都没这问题。

PCI-E 只有X2

转接卡或插口不稳，拔出来清洁一下接口，再插紧一点即可。也有可能是因为转接卡有虚焊，这就需要跟商家沟通一下了，给你换一个或怎么处理了。

2.4G wifi不能连接的问题 (也有人说没有遇到过这个问题)

作者补充：没办法忍受2.4G wifi不能连接的问题，在公共场合大多又是2.4G，好难受，最后闲鱼入手了一台2015年的，换上去一切正常。 弄到最后，感觉自已这样折腾有点主次不分了。。。哈哈。。。

无论5G还是2.4G的WIFI都无法正常联接，经常断连，这是因为PCI-E 2.0是5GHZ频率，刚好跟WIFI冲突了（PCI-E 1.0 是 2.5GHZ），打开后盖就没这问题，因为硬盘发身出来的信号并不强，干扰不厉害。但合上盖子之后，由于WIFI信号引到机身了，导致干扰非常严重。解决办法就是用锡纸屏蔽掉硬盘的信号，但还是只能连5G，2.4G还是连不上，不过没关系了。做法是先用绝缘胶布在转接口处和SSD的接口包一层；然后用锡纸包两层，因为锡纸是导电的，所以千万不要让芯片短路，锡纸可以用五金店卖有粘性的那种，也可以直接用烧烤时用的那种，没什么区别；最后再用纸缘胶布把锡纸包一层，以免漏电。包的时候不要弄得太厚，合不上盖就不好。
