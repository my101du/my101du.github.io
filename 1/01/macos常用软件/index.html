macOS常用软件

permalink: macos-software tags:

  • 工具

macOS（前身是 OS X）是非常好用的操作系统，同时还有很多工具和软件能提高我们的工作效率。

开发人员工具箱

iTerm

OS X 默认的终端功能比较弱，需要下载安装 iTerm2 解锁更多神技(最新的 beta 版本是 3.x 了，但名字还叫 “2” ~.~!)， 点这里 下载iTerm2，或者直接使用 brew cast安装

$ brew cask install iterm2

修改颜色主题

下载所有主题 s

打开 Profile Colors， Import ..选择 Schemes 里面的文件

因为深色主题在频繁切换到其他窗口（白色）的时候，眼睛很累。所以用浅色的。

但是浅色主题在使用 robbyroll on-my-zsh 的时候，出现提示符的路径栏底色黑，字体也黑看不清的情况。

尝试多次，最合适的浅色主题

MacVim——增强版的 Vim

Evernote

使用搜狗输入法的时候，出现中英文混输时，回车想输入英文结果变成换行 关闭搜狗输入法的“自动在中英文之间插入空格”

半角引号被自动替换成全角引号 进入系统设置——键盘——文本，关闭掉“使用智能引号和破折号”

办公

Office

office 2016 节省空间 进入“应用程序”，在 Word 上点击右键，选择 显示包内容，找到 Contents/Resources/Fonts目录 ,把里面的字体文件全部移动到 /Users/xxxxx/Library/Fonts（这个目录默认是隐藏的，进入方法：按住cmd+shift+G，输入 /Users/xxxxx/Library/FontsContents/Resources/Fonts）

按同样的方法重复操作 Excel、Powerpoint 几次，把每个软件的字体都移动出去就可以了。

还有拼写检查文件 SpellCheck 我也删了，不知道有什么后果没有。这些文件放在 Contents/Shared Support/Proofing Tools 里面，只留 English 开头的即可

在安装的时候，选择需要的即可

WPS 出 mac 版本了！ 轻度使用可以替代

只是简单的浏览文件，wps 足够了，打开文件速度很快。

但由于一些兼容问题，如果需要 excel 来处理数据，还是尽可能用原版。

Outlook

Mac 的 outlook 经常出现弹出输入密码，并且反复输入的问题。

即使在 outlook 里删除账号再添加也没有效果。

进入 keychain access, 点击login password，找到 Exchange 或者你集团邮件的地址，删除，重新登录。

默认会把所有历史数据（附件等）都下载，导致公司的邮箱添加后，占用了磁盘8G 空间。。。

在 account -> advance 里勾上这个后，就不会了。

自带 Mail

2018-11-27 更新： 最新版 10.14 里的邮件已经解决了

  1. 复制粘贴图片在邮件正文中间，会自动变成“附件”的问题。
  2. [待测]使用一段时间后，内存占用问题。 如果邮件很多下载完成后，内存占用还是会增加，不久会自动回落。

但是使用中的那种流畅感是 Outlook 无法比拟的。

这里的设置：

经过测试， （1）图片变附件问题，不稳定，有的还是作为正文，有的又变成了附件

默认会把附件下载，导致占用空间很大的问题i

即使设置了 download attachments 为 none（不自动下载，手动下载）

还是会大量下载，导致占用磁盘空间（可能是Exchange 类型邮箱的一个bug，设置后也无效）

可以进入这个目录，删掉所有附件

/Users/YourUsername/Library/Containers/com.apple.mail/Data/Library/Mail Downloads

然后再设置为 “None”

新的 10.14 里这个目录是空的了

可以用这个方式，在菜单里删除。

但是这样好像会把服务器上的附件也给删了。。。（https://www.fonepaw.com/eraser/remove-mail-attachments-mac.html）

所以到这个目录，手动逐个删除

~/Library/Mail/V6/

里面有几个文件夹，找到最大的那个进去，例如 A9970FE6-BB0D-1334-8192-0D7EEasdfa84E
