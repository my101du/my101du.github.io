Magento 是一款国外的商城系统

下载

下载地址 https://www.magentocommerce.com/download

可能是由于家里的网络原因，直接下载压缩包一直失败，翻墙也会中断（也不能复制链接直接通过迅雷等来下载），只好通过 Composer 方法来安装了。其实如果能下载到那个带有测试数据的完整安装包最好，有测试数据学习起来会更方便一点。

按这个页面的说明 http://devdocs.magento.com/guides/v2.0/install-gde/prereq/connect-auth.html

进入 https://www.magentocommerce.com/magento-connect/customer/account/

在 Developer 里生成一个 Secure Key，记住 Public Key 做用户名，Private Key 做密码，后面会用到。

然后先全局翻墙（你懂的，这破网络）运行下面的命令(当然首先要安装 Composer)，按提示输入前面的两个 Key

$ cd Your-Project-Folder
$ composer create-project --repository-url=https://repo.magento.com/ magento/project-community-edition my_magento2

在安装过程中提示你是否需要保存账号信息，这里输入 yes，这样以后就不用重复输入这两个 key 了。 安装完毕后，如果提示错误，例如我这里就提示没有 mcrypt 、intl 两个扩展。于是添加上去(brew 是个好东西！)

$ brew install php56-mcrypt php56-intl
# 然后重启 php 服务加载新的配置文件
# 最后删除旧目录，重新运行 composer 命令

如果报错类似

[UnexpectedValueException]
The checksum verification of the file failed (downloaded from https://repo.magento.com/arch
ives/magento/module-email/magento-module-email-100.0.2.0.zip)

需要先运行 composer clear-cache清空 cache，然后重新运行
