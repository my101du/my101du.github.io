openvc

例子

找图片里的相似区域

使用 TemplateMatching和minMaxLoc 来实现

https://docs.opencv.org/trunk/d4/dc6/tutorial_py_template_matching.html

https://stackoverflow.com/questions/7853628/how-do-i-find-an-image-contained-within-an-image

详细原理（滑动搜索小图，在模板-待搜索图）

https://docs.opencv.org/2.4/doc/tutorials/imgproc/histograms/template_matching/template_matching.html

问题

  • 是否一定是从模板图中截取的相同缩放比例？ （例如我的 logo 是 300x200, 模板图示是500x300, 并不是从模板中截取出来的）

(好像和小图片的大小有关，如果大小和要匹配的区域不一致，就找不到) https://stackoverflow.com/questions/28102385/can-template-matching-in-opencv-deal-with-two-same-sized-images

其他语言

python （推荐）

stackoverflow 上有些 cv 的资料很老了，新版的 opencv 都用 cv2 了

pip install opencv-python

https://opencv-python-tutroals.readthedocs.io/en/latest/py_tutorials/py_imgproc/py_template_matching/py_template_matching.html?highlight=matchtemplate

php

需要安装扩展（因为 opencv 底层是 c++） https://phpopencv.org/index.html

资料 https://medium.com/@morozovsk/computer-vision-and-machine-learning-in-php-using-the-opencv-library-3131fe9df94b

node

支持 3.x 版 https://github.com/justadudewhohacks/opencv4nodejs#readme

只能支持 2.x 版，比较旧 https://www.npmjs.com/package/opencv

npm install --save opencv4nodejs

以及与 Electron 一起使用

https://github.com/justadudewhohacks/opencv-electron
