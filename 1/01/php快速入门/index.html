PHP（外文名:PHP: Hypertext Preprocessor，中文名：“超文本预处理器”）是一种通用开源脚本语言。语法吸收了C语言、Java和Perl的特点，利于学习，使用广泛，主要适用于Web开发领域。PHP 独特的语法混合了C、Java、Perl以及PHP自创的语法。它可以比CGI或者Perl更快速地执行动态网页。用PHP做出的动态页面与其他的编程语言相比，PHP是将程序嵌入到HTML（标准通用标记语言下的一个应用）文档中去执行，执行效率比完全生成HTML标记的CGI要高许多；PHP还可以执行编译后代码，编译可以达到加密和优化代码运行，使代码运行更快。

变量作用域

函数内部是无法访问外部变量的作用域的。

$outside = 'abc';

function foo(){
    $outside = 'efg';
    
    echo $outside;
}

foo(); // 输出  efg

函数访问外部变量，要么把外部的变量设为 global，

$outside = 'abc';

function foo(){
    global $outside;
    
    echo $outside;
}

要么作为参数传递进去

$outside = 'abc';

function foo($outside){
    //$outside = 'efg';
    
    echo $outside;
}

foo($outside); // 输出 abc

PHP 函数

数组

$people = array("Bill", "Steve", "Mark", "David");
echo current($people) . "<br>"; // 当前元素是 Bill
echo next($people) . "<br>"; // Bill 的下一个元素是 Steve
echo current($people) . "<br>"; // 现在当前元素是 Steve
echo prev($people) . "<br>"; // Steve 的上一个元素是 Bill
echo end($people) . "<br>"; // 最后一个元素是 David
echo prev($people) . "<br>"; // David 之前的元素是 Mark
echo current($people) . "<br>"; // 目前的当前元素是 Mark
echo reset($people) . "<br>"; // 把内部指针移动到数组的首个元素，即 Bill
echo next($people) . "<br>"; // Bill 的下一个元素是 Steve
print_r (each($people)); // 返回当前元素的键名和键值（目前是 Steve），并向前移动内部指针

array_map array_filter array_reduce 区别

//filter, reduce array_filter($arr, function($item, $key) {})

//map array_map(function($item, $key) {}, $arr)

  1. map: 类似 foreach 处理每个元素 (例如每个元素求平方) 注意,它的参数顺序是个反的! 先 func, 后 array….
  2. filter: 类似先 foreach 检查某个元素, 然后根据判断条件, 符合条件的元素才返回. 不符合的丢弃.(例如返回数组中的一部分偶数)
  3. reduce: 对每个元素进行相同的操作,然后返回一个值 (归纳, 例如求和. 去重。 可以重复利用里面那个“临时值”)
$d = array_map(null, $a, $b, $c);

合并与去重 array_merge / array_unique

$arr_1 = ['a', 'b', 'c', 'user'=>'eric'];
$arr_2 = ['b', 'd', 'f', 'user'=>'amy'];

$arr_mer = array_merge($arr_1, $arr_2);

['a', 'b', 'c', 'b', 'd', 'f', 'user'=>'amy']

发现有个重复的 “b” !!!
而 “user” 的值已经被替换了!

这是因为 array_merge 只能把 索引为字符串的值合并, 而 索引为数字的则直接添加到新数组.

为了让第二个数组里的 “b” 也被合并

再次用 array_unique()

PHP 7

PHP7 不再支持与类名相同的构造方法，构造方法统一使用 __construct()

例如 new ClassName(‘xxxx’) 可能会报错

Methods with the same name as their class will not be constructors in a future version of PHP; Participant has a deprecated constructor (View: …)

修改类文件，将里面的和类名相同的方法，改成 __construct，如果不方便改的话，只能降低用 PHP5.6 了

一些技巧

使用 compact 函数自动生成指定索引的数组

通常，我们要把一个字符串作为“索引”，把一个数组作为“值”来组成一个新的数组，可以用这端代码，手动指定索引名

$index = "test";
$value = array('a'=>'111', 'b'=>'222', 'c'=>'333');
$new_array = array($index=>$value);

可以使用 compact 函数精简这段代码，省掉手动定义“索引”的步骤，自动从变量名里得到索引。

如下，参数是一个字符串test，但它会去搜寻变量名为这个字符串的数组变量$test，然后把这个字符串作为“索引”，变量值作为新的数组的值。

$test = array('a'=>'111', 'b'=>'222', 'c'=>'333');
$new_array = compact('test');

得到

Array ( [test] => Array ( [aa] => 11111 [bb] => 222 [cc] => 3333 )

)
