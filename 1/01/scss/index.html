SCSS

有各种好处，使用 SCSS 处理器将它转成 CSS

编写 scss 样式文件

从例子来学习, 创建目录和文件

styles base.scss iconfont.scss mixin.scss

首先看 base.scss

/*导入外部文件*/
@import "./mixin";
@import "./iconfont";

/*这些都和 CSS 标准写法一致*/
page, body {
  font-size: 32px;
  height: 100%;
  /*这里有个“变量”  $bg-color-base, 等下看 mixin.scss 文件就明白了*/
  background-color: $bg-color-base;
  color: $color-dark;
  box-sizing: border-box;
}

@media screen and (min-width: 500px) {
  html,body {
    max-width: 1500px;
    margin: 0 auto;
    position: relative;
  }
}

//******常用类******/
.clearfix:after {
  content: '';
  display: block;
  clear: both;
}

.clearfix {
  zoom: 1;

  &:after {
    content: '';
    display: block;
    clear: both;
  }
}

.show {
  display: block;
}

.ellipsis {
  overflow: hidden;
  text-overflow: ellipsis;
  white-space: nowrap;
}

mixin.scss 文件，一般在这里定义变量和方法，例如主题颜色、一些需要动态计算的尺寸等

变量，以 $开头 方法，以 @mixin 开头，后面跟着方法名。如果有参数的话，参数是变量的形式，可有一个或多个参数，参数也可以有默认值。实现用一个方法名处理一系列 CSS 规则

/* colors
-------------------------- */
$color-primary : #D87182 !default;
$color-dark : #333 !default;
$color-dark2 : #666 !default;
$color-light: #888 !default;


$btn-disabled-1: mix(#fff, $btn-active, 70%) !default;
/* 53a8ff */


/* methods
-------------------------- */
// 背景图片地址和大小
@mixin bg($url) {
  background-image: url($url);
  background-repeat: no-repeat;
  background-size: 100% 100%;
}

// 定位上下左右居中
@mixin center {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}

// 宽高
@mixin wh($width, $height) {
  width: $width;
  height: $height;
}

// 字体大小，颜色
@mixin sc($size, $color) {
  font-size: $size;
  color: $color;
}

// flex 布局和 子元素 对其方式
@mixin fj($type: space-between) {
  display: flex;
  justify-content: $type;
}

使用

导入 base.scss 基础样式

import './styles/base.scss'

覆盖，或者增加一些在特定页面的样式。

html 代码

<div class="home-page">
    <class="nav-list">
        <div class="nav-item"></div>
    </div>
    <div class="recommend">
        Some Text
    </div>
</div>

添加文件 index.scss，内容

@import "../../styles/mixin";

.home-page {
  @include wh(100%, 100%);
  .nav-list {
    @include fj(center);
    flex-wrap: wrap;
    margin-top: 20px;
    .nav-item {
      width: 23%;
      margin: 0 5px;
    }
  }
  .recommend {
    display: block;
    width: 100%;
    text-align: center;
    margin: 30px 0;
    font-size: 36px;
  }
}

注意到先导入了 mixin.scss, 然后才能使用里面的变量和方法。

@include 方法名(方法参数1, 方法参数2.。。)

相当于调用了 mixin.scss 里对应的方法返回的 css 规则。

另外注意到两个选择器 .home-page 和 .recommend 是“嵌套”的。 这样就不用像 css 这样写了 .home-page .recommend 。
