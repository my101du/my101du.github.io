思维体系的几个维度

要理解，它能帮助形成清晰的思维，归纳概括、综合分析、提出对策。

从不同的角度切入，或者从不同的角度结合

写不出东西，不是背诵的范文不够多，而是脑中没有“框架”和“思维”工具

关键就是用好“辩证法”，包括

  1. 两分法： 多角度看问题
  2. 结合式： 交叉性思维，在不同角度之间找内在联系、结合点。能够自如地把不同的角度按照不同标准进行组合、融为一体。
  3. 吃透各个逻辑脉络的 内涵 + 外延， 熟悉 具体表达 和 使用范围， 具体问题具体分析，最后把思维语言->表达语言，并作出适当的引申。

后面有例子

实务维度

政府实务主要体现在以下，涵盖了政府工作的所有内容

  • 主体： 包括：政府，企业，民众 主体的方面： 利益，思想，素质
  • 技术： 硬技术：技术手段，技术路径（方法），人，财，物 软技术：管理手段，方式，流程
  • 制度： 正式制度：规则，机制，体制（政策） 非正式制度：风俗，习惯等
  • 环境： 自然地理环境 社会人文环境

（我自己想象的：一个人要做一把椅子， 主体：首先脑子里想，我要做什么，为什么做，能不能做，划不划算； 技术:接着是要想用什么材料，用什么工具砍树，怎么砍，去哪里砍更省力，要砍多少树 制度：然后砍树回家，看看做椅子要哪些必须要遵守的工序，或者一些参考的资料，别人是怎么做的。 环境：做完椅子，是不是可以坐着安静地喝茶，精神和身体得到满足，卫生是不是搞完了。。 ）

主体

主体不明确，谈制度、技术都是空话。 实际情况中要把 政府、企业、民众这三个抽象的主体具象化、具体化到涉及的实际问题中。

分析主体，从几个方面：

内在需要（利益）

  • 经济利益 人们对钱的需要
  • 政治利益 权利和权力 公权力和私权利
  • 文化利益 精神的需要
  • 民生利益（社会利益） 人们对就业、社会保障、教育、医疗、住房、公共安全等的需要
  • 生态利益 对美好生态环境的需要

内在思想

  • 理念 宏观层面的思想：理想，信仰，战略层面的认识。如为人民服务，执政为民，树立共产主义、集体主义、公平公正、珍爱生命等 （平时要多积累，不要空话套话）
  • 意识 和理念的不同->往往有个具体的事承载。如交通安全意识，食品安全，环境保护，城市安全，用药安全
  • 知识（常识） 微观层面，很具体，如 抗生素案例->树立用药安全的意识->要具备用药安全的知识（强调具体的层面）， 写作时可以和“意识”综合运用（有具体认识 – 树立意识）

外在素质或行为

  • 业务素质 工作能力 工作素质
  • 思想道德素质
  • 心理素质
  • 身体素质等等
  • 外在行为 例如有的监管部门不作为

技术

硬技术

技术路径： 各种科学技术和技术手段，如监控、天气/地震预报、网络、信息技术等

人：管理具体事务的人力投入（高频词：人才，队伍，团队，干部，编制等）

财：经费投入 政府财政投入（如财政预算决算） 企业资金投入（比较流行的经费来源： 民间资本，民间融资-开放能源金融领域，让民间资本进入铁路的运营和管理等等 BT(建设-移交)：用非政府资金进行基础非经营性设施建设项目 银行贷款、投融资平台等 公益慈善渠道经费来源

物：工作事务的物质投入，如设备、设施，机构 设备：机器、车辆等 设施：基础设施、配套设置。 不仅包括公路铁路机场通讯水电煤气等公共设施， 还包括教育科技医疗卫生体育文化等“社会性基础设施” 配套设施是指城镇道路、市场、供水排水、供电、邮政、卫生、环保等（与公共设施有关）；还有学校、医院等机构（与人有关）

软技术

管理：管理的规划、执行、监督和评估，还包括监管。 四个要素： 管理主体-由谁管；管理客体-管什么；组织目的-为何管；环境条件-什么情况下管

制度

正式制度

规则(规定) 静态的，如村民自治规定等。

机制 哪些部分组成，为什么由这些部分组成；怎样工作，为什么这样工作—> 协调各个部分之间的关系的运行方式。是 动态 的。

经常用到的：

  1. 科学的决策 : 民情反映，社会公示，社会听证，专家咨询，决策论证，责任制。。。
  2. 预防应急：组成应急小组，指定工作计划；确定联络方案，保障信息畅通；开设热线电话；收集各方咨询；协调有关单位共同开展工作
  3. 组织协调：
  4. 监督落实
  5. 总结反思： 吸取教训，交流经验

体制（政策）

是一个宏观的概念。当前最大的体制性问题：政府职能问题->多头管理，政出多门。 千万不要去讨论政治体制。。。什么民主xxx. 政治性敏感问题，小心零蛋。。

非正式制度

风俗，“潜规则”等

环境

自然地理环境： 例如分析渤海。。。的原因： 有一个就是自然地理条件所限-半封闭内海

社会人文环境： 例如现在很多农村人觉得读书不如打工-就是社会整体的浮躁氛围。 加大宣传力度，营造……良好氛围。

从时间维度

用 动态 和 发展 的眼光看问题

高频词： 历史与现状，传统与现代。（例如概括材料，讲 80年代。。。90年代。。。21世纪。。。）

微观：一件事、工作进行的不同过程：事前、事中、事后。

例如，要解决某个问题、矛盾，可以进行这三个阶段的管理。 事前： 危机预测、预防、准备：核心是做好预警、防范和准备工作，为政府主动公关奠定基础 事中： 危机爆发后的应对管理 时候： 危机平息后的总结和恢复 （回忆到，前天看过的一则“建立城市应急体系”，也是从这三个方面分析的）

（个人理解：上一个做椅子的案例：

  1. 传统的椅子是怎样的，但时代变了，我要做一个现代风格的椅子
  2. 为了把这个椅子做好，要先把图纸画好，木头的长度算好；做的时候要细心认真，按时完成；做完后如果有问题要总结，有经验与同行分享下 ）

空间维度

物理空间： 国内国外的类似案例对比，处理方式对比，拓宽思维视野。

思维空间：内因 & 外因（事物的内部矛盾和外部矛盾，内因决定外因， 外因通过内因起作用， 本-标 的关系）

案例：标本兼治： 解决三农问题， 外部输血（外因作用），内部造血（内因的作用）

（个人理解： 造好这把椅子，除了自己要态度认真、充分准备，还要充分学习村里面、别的国家同行的经验，从其他行业、传统文化的设计里汲取营养，才能造出好椅子； 另外，也不能完全照抄，那样会失去个性，还要融入自己的想法 ）

价值维度

其实就是辩证思维，来表达 利与弊 ，成绩与问题，经验和教训，意义和危害。

（个人理解： 这把椅子造出来，我能获得什么好处、有什么损失； 做好了能让自己多一门技能，但是可能浪费时间等。。。。 乱讲的。。。 ）

政府，企业，百姓。没错，但是不具体、不准确。 （“政府”，“有关部门”是模板式表达，要找到具体的政府部门，如食品质量监督局，工商局，质监局； “企业”虽然无法点名道姓，但是要说清楚哪些种类的企业。）

时间维度分析

真题案例

从一篇材料中，F市实施心理健康促进项目的工作大事记，进行分类总结

关键 将多个维度里的不同角度 如主体（制度、机制、体制）技术（人，财，物，监管）等等高频词，将采分点像放入文件夹一样进行整理出来。

同时注意同类的进行合并，不同的进行罗列，相关的关键词合并成同类项。(例如 心理健康辅导中心、学生个体心理辅导室、心理咨询中心。。。概括成“各类心理辅导中心” )

回答技巧： 多条罗列，每条里面还可以按 总-分 式再次展开，并且多个“总”的格式能相似

  1. [主体] 政府重视，社会参与。 [展开前面的总论点]副市长挂帅成立领导小组；工会、妇联、残联、学校、企业广泛参与。
  2. [制度] 建立制度，协调落实。（注意和前面的总论格式一致，4个字） [展开，什么制度，怎样协调、落实] 制定工作方案，建立会议部署、每月汇报等工作协调机制，打造社会心理健康促进机制
  3. [技术] 增加投入，完善保障。 [财]提供财政保障；[物]建立各类心理咨询室；[人]聘请和培养心理咨询师等专业人才；[技术手段]成立心理卫生协会；打造心理干预服务模式，建立咨询热线
  4. [主体-思想-意识/知识，环境-社会人文] 教育宣传，营造氛围。通过主题日、广场活动、报纸专版等方式加大宣传力度；通过开设课程和知识培训加强教育

注意

考察的是辩证思维，以及思维的综合、连贯性和思想的深刻性，不是简单的议论文，也不是简单地写作文。

写作能力的高低，对分数影响不大。（经过 x40% 后大部分人都在 20-25分。。。）

概括题不要失分。写作差距不大

不要背诵，背诵可能会限制思维定式，同一个主题，可以从不同的角度切入。（例如"垃圾治理”，从公民素质和社会矛盾都可以。） 题目往往是“主题意料之中，角度意料之外”。 例如这几年“文化”主题，有时候从精神层面，有时候是市场背景下的思想道德状况，有时候是文化的载体层、文化遗产保护（近来有意回避热点素材，选思想文化，生态文明）

不是个性化，不要用华丽的辞藻和旁征博引， 名言警句在合适地方点题就好 ——平和，中庸，沉稳。

《南方周末》化 NO! 杂文。。 xx 问题，xx愤怒，xx岂止。。。确实想起了南方的地方新闻台。。 《人民日报》化 YES！ 一盘棋，有问题，找根源，要克服。

立意要新，但不是奇特。

一定要基于材料，利用好材料，但不要照抄材料 在就事论事中，要上升到一定的高度。 例如 幸福-中国梦，水电站-生态文明 充实 -> 立足材料，联系实际的内容填充框架 精炼 + 表达规范，句子控制在 20个字内

阅卷老师角度

“看论点”：没有在开头或最明显的地方出现，后面结构、分析再好，都属于“没有观点” “看结构”：实际上只是看每段的第一句话。 “看标题”：是否精炼概括 “错别字”：会产生不良印象

客观题因为给分点明确，所以要争取不要丢分。 主观题，不愿意给了高分惹麻烦，往往区分没那么大。

->努力抓好前面两题，找采分点，比努力练大作文要有用。

条理 过渡 要明显、清楚

字数：标点符号 空格 都是算字数的。 字数不够，材料凑；字数太多要割舍

卷面： 整洁的卷面->印象好->内容看得清楚->分数 +5/10 字：没时间练字，可以不漂亮，但绝对不能潦草。 关键的地方（第一段，最后一段，每段第一句，绝对不要涂改）
