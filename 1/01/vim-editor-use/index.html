Vim编辑器的使用教程

  Vim 技能  评论 字数统计: 1632字 阅读时长: 4分

编辑器之神与神的编辑器，Vim 是一个跨平台、高效率的编辑器

使用 Vim 替换 vi

centos 下

yum inistall vim-common

vi ~/.bashrc

#添加别名
alias vi="vim"

Mac 下用 MacVim 代替系统自带的 vim

因为 YouCompleteMe 插件需要 7.3.598 以上的版本，而 Mac 自带的版本较低。 但是我们又不想替换或删除系统自带的 vim（因为可能会带来一些风险），所以只要用“别名”的方式，让 vim 默认调用 MacVim 了。

# 编辑 ~/.profile
alias vim='/Applications/MacVim.app/Contents/MacOS/Vim'

source .profile

键盘基本操作

ESC 在不同模式之间切换

Ctrl+F 往下翻页（forward 向前） Ctrl+B 往上翻页（back 返回） 方向 H J K L （左 下 上 右）

shift+v，再按方向键 选中多行

dd 剪切选中的多行/或当前行 yy 复制当前行 y 复制选中的多行 p 粘贴 i/I 在当前光标前面/在当前行最前面 进入编辑状态 a/A 在当前光标后面/在当前行最末尾 进入编辑状态 r[字符] 把当前的字符用另外的字符替换掉 x 删除当前的字符 u 撤销上一步操作，可一直重复

[数字]G 跳到行号所在的行 gg 跳到文档顶部 G 跳到文档末尾

/[字符串] 搜索字符串，按n在多个搜索结果中前进 :%s/[原字符串]/[替换字符串] 替换字符串

:wq 保存,退出，加”!“表示强制

配置文件常用参数

一般配置文件位于 ~/.vimrc

代码高亮

syntax on

放在第一行

一般通过正在编辑的文件后缀名来处理高亮，如果一些 shell 脚本没有后缀（例如 git hook 的 post-receive）,需要在这些文件顶部添加 #!/bash/sh

#!/bash/sh

cd ~/Downloads/

编码支持

# 支持中文显示与自动适配 GBK 等中文编码问题
set encoding=utf-8
set fenc=cp936
set fileencodings=cp936,ucs-bom,utf-8
set nobomb

# 显示行号
set nu

# tab 宽度
set tabstop=4 softtabstop=0 noexpandtab shiftwidth=4

技巧

多行缩进

shift + V 进入块模式 h j k l 选择多行 > 缩进（shift + ,）

多标签同时编辑多个文件

基本命令

进入vim前 vim -p <文件名> 以多标签形式打开文件

标签操作

:tabnew 打开一个新标签 :Ex 打开文件浏览器，选择文件进行编辑 :tabnew <文件名> 直接在新标签打开一个文件进行编辑 :tabf <部分文件名> 在当前目录下根据部分文件名自动匹配 :tabs 显示已打开标签页的列表，并用“>”标识出当前页面，用“+”标识出已更改的页面。 :tabc 关闭当前标签页 :tabo 关闭所有的标签页 :tabn或gt 移动到下一个标签页 :tabp或gT 移动到上一个标签页 :tabfirst或:tabr 移动到第一个标签页 :tablast 移动到最后一个标签页 :tabm [次序] 移动当前文件的标签页次序。比如:tabm 1将把当前标签页移动到第2的位置。如:tabm不指定参数将被移动到最后 :set showtabline=[1,2,3] 标签页在窗口上方显示标签栏。=0完全不显示标签栏，=1只有用户新建时才显示，=2总是显示标签栏 :tabdo <命令> 同时在多个标签页中执行命令。比如:tabdo %s/food/drink/g 命令把当前多个标签页文件中的“food”都替换成“drink” :help tab-page-intro命令，可以获得关于标签页使用的更多信息

设置快捷键，替换 :tabp 和 :tabn 快速在标签中间前进后退

# 编辑 ~/.vimrc 
map <S-Left> :tabp<CR>
map <S-Right> :tabn<CR>

让标签隐藏路径，只显示文件名

如果文件所在的路径较深，标签上为了节省空间会显示奇怪的类似 /t/u/X/filename.ext，实际上我们从文件名就能看出它是干什么的，前面这一堆目录首字母缩写反而影响了工作，需要隐藏掉。

编辑 ~/.vimrc，在后面加上如下一段内容

set tabline=%!MyTabLine()

function MyTabLine()
  let s = ''
  for i in range(tabpagenr('$'))
    " select the highlighting
    if i + 1 == tabpagenr()
      let s .= '%#TabLineSel#'
    else
      let s .= '%#TabLine#'
    endif

    " set the tab page number (for mouse clicks)
    let s .= '%' . (i + 1) . 'T' 

    " the label is made by MyTabLabel()
    let s .= ' %{MyTabLabel(' . (i + 1) . ')} '
  endfor

  " after the last tab fill with TabLineFill and reset tab page nr
  let s .= '%#TabLineFill#%T'

  " right-align the label to close the current tab page
  if tabpagenr('$') > 1 
    let s .= '%=%#TabLine#%999Xclose'
  endif

  return s
endfunction

function MyTabLabel(n)
  let buflist = tabpagebuflist(a:n)
  let winnr = tabpagewinnr(a:n)
  let label =  bufname(buflist[winnr - 1]) 
  return fnamemodify(label, ":t") 
endfunction

然后运行下面的命令即可

source ~/.vimrc
# 进入 vim 后，再次运行
:source ~/.vimrc

插件

YouCompleteMe

安装过程比较曲折，例如 brew 的权限，需要添加库等等。

PHP 自动完成

https://github.com/shawncplus/phpcomplete.vim

使用 Vundle 安装

Plugin 'shawncplus/phpcomplete.vim'

用更强大的 SuperTab 替代 c-x，c-o

参考 https://jordaneldredge.com/blog/setting-up-php-completion-in-vim/

autocmd FileType php setlocal omnifunc=phpcomplete#CompletePHP
set completeopt=longest,menuone
Bundle 'ervandew/supertab'
let g:SuperTabDefaultCompletionType = "<c-x><c-o>"
Bundle 'shawncplus/phpcomplete.vim'

MarkDown

https://github.com/plasticboy/vim-markdown/

用 Vundle安装

# 编辑 vimrc，添加
Plugin 'godlygeek/tabular'
Plugin 'plasticboy/vim-markdown'
# 运行
:so ~/.vimrc
:PluginInstall

关闭折叠

let g:vim_markdown_folding_disabled=1

其他配置参数参看 github

i 光标前编辑 I 在行首 a 光标后编辑 A 行尾编辑

:w 保存 :q 退出 :wq! 保存、退出(强制) :%s/源字符串/目的字符串/g
