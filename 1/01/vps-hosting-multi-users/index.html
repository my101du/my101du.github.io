一键包 & 面板

Lnmp 一键包

安装很慢，升级不便（据说编译安装对本机配置进行优化？） http://forum.ubuntu.org.cn/viewtopic.php?t=247502

是否 lnmp 一键包 省空间一点？ 1.2G -> 1.6G

现在建议用 ubuntu 的 apt-get 安装

WDCP

宝塔

一些功能只有专业版有（木马查杀 等） https://www.bt.cn/download/linuxpro.html

如无特殊： Nginx 建议最新版本， PHP 7.1， MySQL5.5（小于1G低内存）

下载安装脚本，根据提示安装依赖的库和宝塔程序

https://www.bt.cn/bbs/thread-1186-1-1.html

yum install -y wget && wget -O install.sh http://download.bt.cn/install/install.sh && sh install.sh

如果提示下面这个，说明前面的安全策略没有打开 5880 端口（终于知道这个干什么用的了。） 还有个 888 端口，好像也有用。。。

安全组出方向要放行5880端口 不然将安装不上面板!

–2018-08-27 16:53:17– (try: 4) http://125.88.182.172:5880/install/src/pip-9.0.1.tar.gz Connecting to 125.88.182.172:5880… failed: Connection timed out. Retrying.

然后登陆到宝塔面板正式安装 Nginx 等应用

似乎 Ubuntu 无法使用极速安装（过程很久），CentOS 使用 RPM 方式？

因为 PHP7.2 对一些函数支持的变化，可能导致部分应用无法运行，另外压测里变化并不是极大，因此我这里选 7.1 足够了。 MySQL 的话，5.6 和 5.7 对系统占用资源较大，建议 1G 以上内存的服务器使用。

重点：MySQL5.6中内置了对Innodb的全文索引支持 innodb内部性能增强

CentOS7 极速安装非常快

腾讯云设置安全组

官方： https://www.bt.cn/bbs/thread-1229-1-1.html

入站默认拒绝所有

添加入站规则（这里没有按官方的添加ftp等端口，因为我只用 sftp 足够）

出站默认拒绝所有，所以要添加出站的规则，把入站规则的复制进来就好

然后切换到“关联实例”，把规则绑定到云主机

注意，如果需要在腾讯云上登录其他服务器（跳板），需要开启 “出站” 22

http://www.xmirror.cn/

官网首页看上去很不靠谱啊。。 https 错误，时间截止到 2017 年

登录必须要 root 账号 坑爹文档没说，其他用户名在 root 组都不行

开启下6361端口，防止登录失败。

# install
sudo wget -O install.sh http://dl.xmirror.cn/a/install.sh && sudo sh install.sh

# 安装时选 Proxy install 模式，用 Normal install 会提示下面的错误

Install Xmirrorsec for Nginx: [ FAIL ] Install Xmirrorcc for Nginx: [ FAIL ]

注意如果 Nginx 用了 1.8 版要切换回来1.12，否则会提示

heck Web Server: [ NOT FOUND ]
