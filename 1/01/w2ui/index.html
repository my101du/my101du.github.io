或者来自 json 文件

除了在开始的 sample 里直接赋值数组数据到 grid, 还可以通过 url 参数从服务器请求数据（ajax, json 文件等）

$('#myGrid').w2grid({ 
    name   : 'myGrid', 
    url    : 'data/records.json',
    columns: [                
        { field: 'fname', caption: 'First Name', size: '30%' },
        { field: 'lname', caption: 'Last Name', size: '30%' },
        { field: 'email', caption: 'Email', size: '40%' },
        { field: 'sdate', caption: 'Start Date', size: '120px' },
    ]
});

请求服务器上的数据

在 grid 内部，会产生如下的 数据结构 的 POST 请求

{
    "cmd"         : "get-records",
    "limit"       : 50,
    "offset"      : 0,
    "selected"    : [1, 2],
    "searchLogic" : "AND",
    "search": [
        { "field": "fname", "type": "text", "value": "vit", "operator": "is" },
        { "field": "age", "type": "int", "value": [10, 20], "operator": "between" }
    ],
    "sort": [
        { "field": "fname", "direction": "ASC" },
        { "field": "Lname", "direction": "DESC" }
    ]
}

因此 grid 会期望返回的 json 数据是如下结构

{
    "status"  : "success",
    "total"   : 36,
    "records" : [
        { "recid": 1, "field-1": "value-1", ... "field-N": "value-N" },
        ...
        { "recid": N, "field-1": "value-1", ... "field-N": "value-N" }
    ]
}

如果并非上面的结构，你可以在 grid 的 onRequest 和 onLoad 事件中进行数据转换

如果返回错误，则是下面的结构

{
    "status"  : "error",
    "message" : "Error Message"
}

删除记录

当用户多选了记录，并且点击 toolbar 上的 删除 按钮，则会触发 .delete() 方法, grid 组件发送请求

{
    "cmd"      : "delete-records",
    "selected" : [1, 2],
}

会返回正确或错误结果（错误的json 和前面一样）

{
    "status" : "success"
}

修改、保存数据

如果编辑了 grid 里面的数据，则会发送

{
    "cmd"     : "save-records",
    "changes" : [],
}

Utilities

在 w2ui 库中有些全局变量，并且有自己的方法

可以对数据进行 data validation, data encoding, transitions

w2ui

所有创建的对象 (layouts, grids, toolbars, sidebars, tabs, forms) 都会被附加到 w2ui 全局对象上

可以访问它们，或者覆盖它们

w2ui['myGrid'].get(3); // returns record with id=3
// or
w2ui.myGrid.get(3);

w2obj

如果需要添加自定义的方法给 layout/grid 等的 prototype. 可以使用 w2obj

w2obj.grid.prototype.myFunction = function () {
    console.log("This function exists in all grids. ", this);
}
;

w2utils

有一些通用函数进行数据验证、转码、清理等

w2utils.isEmail('som@email.com'); // returns true if the string is a valid email

methods 全部的方法
