Flutter 简介

跨平台， React Native, Xaramin 等

Dart

对中国用户，需要修改一些地方 https://github.com/flutter/flutter/wiki/Using-Flutter-in-China

一些评论：

https://www.v2ex.com/t/434807#reply36 总的来说就是坑多

包下载

https://pub.dartlang.org/packages

安装

下载 SDK，解压后指定环境变量

https://flutter.io/setup-macos/#get-sdk

解压 SDK zip, 把目录flutter改名为flutter-sdk,复制到某个目录，例如 ~/Document/flutter-sdk。

最后修改环境变量

vi ~/.bash_profile

# 添加
export PATH=~/Documents/flutter-sdk/bin:$PATH

source ~/.bash_profile

检查问题

flutter doctor

按提示安装缺失的工具和库

可以安装 Android Studio, 或者 VSCode 的插件支持

最新的 Android SDK, 无法直接通过命令行（android和tools/bin/sdkmanager）直接打开 sdk manager 了，必须使用 Android Studio 来管理

Android Studio 启动时，提示要修改 Proxy (FQ)才能正常下载，选 Socket, 127.0.0.1 端口 1080

升级与版本选择

查看当前SDK版本,可以切换 beta/master 版本，建议使用 beta 版以尽快解决一些问题。

# 查看
flutter channel

# 切换分支
flutter channel beta
flutter channel master

在项目中指定版本，修改pubspec.yaml文件

name: hello_world
dependencies:
  flutter:
    sdk: flutter
dev_dependencies:
  flutter_test:
    sdk: flutter

运行flutter doctor如果出现下面的提示，可以升级

在项目中升级 Flutter SDK 和包

flutter upgrade

# 查看当前依赖，以及最新版本
flutter packages get
flutter packages upgrade
