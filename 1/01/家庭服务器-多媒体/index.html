xmbc/kodi 影院/剧集

中文

“interface”

首先修改字体为 arial base

然后修改界面 language

如果找不到： 选择System（第3步图中右下角），进入此界面，选择Add-ons，再点击Running1 Install form repository Look and feel2 H Languages$ j 选择简体中文，在这里会下载 返回设置界面，重复第3步。进入Interface后选择Regional，点击Language，选择简体中文即可

远程播放

点击文件， 浏览， 选择 zero config , 选择 NFS/SMB 。。。

如何删除之前添加的垃圾文件夹？

  1. 设置，视频，清理资料库（好像无效。。。。）

  2. 如果是电脑版，可以双击这个文件夹，弹出删除菜单。 但是 TV 上怎么双击？(长按？)

  3. 直接修改配置文件： 依次进入User-〉Library-〉Preferences-〉XBMC-〉userdata 点击sources.xml，选择Text Viewer 点击左上角的编辑，把包含你目录信息的那一段删除，从删到。储存就可以了

  4. Using google suggests setting the source to but I can’t finagle that option as well.

将 NAS 的网络共享文件夹作为资料库，自动获取海报等

上一个的升级版

  1. 在 NAS 里打开 NFS 共享。
  2. 添加 “豆瓣” 插件，用于获取简介、演员、海报等
  3. xmbc 设置里，添加一个 NFS 类型的共享文件夹。 注意关键 在类型里选择“电影”，信息来源方式选择“豆瓣”
