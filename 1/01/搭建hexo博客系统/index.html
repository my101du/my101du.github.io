一些问题

启动hexo server提示

hexo generate fail - Template render error: (unknown path)

nunjucks syntax issue in theme layout.

原因

The {{}} is nunjucks syntax, therefore please avoid using them in posts.

运行下面的命令也看不到哪个文件有语法错误

hexo generate --debug

查了一些资料 {{}}也不行，要改成 {%%} 但这样就无法在编辑时准确体现代码。。。
