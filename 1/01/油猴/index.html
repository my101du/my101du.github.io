在 ajax 加载的元素准备好后，自动点击

默认只能处理在 window 里加载完毕的元素，假设页面内一部分是通过 ajax load html 的，那么元素是拿不到的。

尝试 window.setTimeout(updateHTML, 2000); 无效。

使用 GreaseMonkey 的 waitForKeyElements (https://gist.github.com/BrockA/2625891) https://stackoverflow.com/questions/8281441/fire-greasemonkey-script-on-ajax-request/8283815#8283815

require 它进来

有时候多步动作，要拆成几步。不然在执行部分后面代码的时候，又来一次ajax 加载，就会出现“属性找不到”问题

// ==UserScript==
// @name         TTI Framework Dev tools
// @namespace    http://tampermonkey.net/
// @version      0.1
// @description  try to take over the world!
// @author       Eric Zhang
// @require  https://gist.github.com/raw/2625891/waitForKeyElements.js
// 匹配该 url 下全部的地址
// @match        http://192.168.56.101:8081/*
// @match        http://192.168.56.101/?fcontroller=tr/test_request&ftitle=Test%20Request
// @grant        none
// ==/UserScript==


function getParameterByName(name, url) {
    if (!url) url = window.location.href;
    name = name.replace(/[\[\]]/g, '\\$&');
    var regex = new RegExp('[?&]' + name + '(=([^&#]*)|&|#|$)'),
        results = regex.exec(url);
    if (!results) return null;
    if (!results[2]) return '';
    return decodeURIComponent(results[2].replace(/\+/g, ' '));
}
