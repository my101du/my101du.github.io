用户权限相关插件

用户组与权限

User Role Editor

增加角色，设置通用权限（例如页面编辑、文章编辑、主题修改等） 但是无法修改后台菜单

菜单查看

Admin Menu Editor

要 Pro 版本才能设置其他角色

添加自定义链接跳转到任意页面

Remove admin menus by roles

(这个有问题，会突然出现设置无效，上面的那个插件修改后会影响它。。。)

先用上面的 User Role Editor 创建角色（首先赋予管理员权限），然后删除部分权限（这个菜单插件只是隐藏，并没禁用权限，要禁止用户点击“更新”等顶部的按钮）

拉到页面底部，勾上 hide admin bar， 避免把一些插件的快捷方式显示出来（例如 The-7 主题设置，更新按钮等）

接着用这个设置显示的后台菜单

(只能一次给多个角色设置统一的菜单，无法分角色)

可以自己写个插件，生成一组后台菜单

最终选择：Adminimize

强大的设置。

注意这个框的意思：

红色： 勾上 表示隐藏 灰色： 勾上 表示隐藏

也就是说，不管什么颜色，勾上就是隐藏。。。

注意：用户能够修改“外观”，必须在 User Role Editor 里打开 “switch theme” 权限才能进入里面的页面，否则不会出现“小工具”等等子菜单
