PHP-HTML-Parser

当前这个 PHP-HTML-Parser 比较流行。

https://github.com/paquettg/php-html-parser

[弃用]最大的问题是，不能像 jQuery 那样支持层级选择器，例如 find('.class_name h2’), 而必须先 find('.class_name’), 然后在内部去找 h2

安装与使用, load()

composer require paquettg/php-html-parser

基本使用（可以在 github 源的 tests目录里看到更多的案例）

load()可以自动根据参数来调用是否 URL 、 File、String 类型

use PHPHtmlParser\Dom;

$dom = new Dom;

// (1)load from html string
$dom->load('<div class="all"><p>Hey bro, <a href="google.com">click here</a><br /> :)</p></div>');

// (2)load from files
//$dom->loadFromFile('tests/big.html');

// (3)load from URL
$dom->loadFromUrl('http://google.com');
$html = $dom->outerHtml;

$a = $dom->find('a')[0];
echo $a->text; // "click here"

// more than 1 elements
$contents = $dom->find('.content-border');
echo count($contents); // 10

foreach ($contents as $content)
{
	// get the class attr
	$class = $content->getAttribute('class');
	
	// do something with the html
	$html = $content->innerHtml;

	// or refine the find some more
	$child   = $content->firstChild();
	$sibling = $child->nextSibling();
}

loadStr()

如果使用 loadStr 加载 String, 则不会进行自动检查。这是由于在某些情况下，如果使用 load方法，因为它会去判断是否字符串、是否 url、文件名，因此当参数值是一个非常大的字符串时，可能会报错。

而直接使用 loadStr则会直接告诉系统，用处理字符串的方式来处理。跳过自动判断过程。

$dom->loadStr('<html>String</html>', []);
$html = $dom->outerHtml;

Options

可以为 $dom设置全局选项

$dom = new Dom;
$dom->setOptions([
	'strict' => true, // Set a global option to enable strict html parsing.
]);

也可以在 load()方法调用时，传入参数

$dom->load('http://google.com', [
	'whitespaceTextNode' => false, // Only applies to this load.
]);

$dom->load('http://gmail.com'); // will not have whitespaceTextNode set to false.

目前有7个选项

  • Strict 默认false 严格模式，html 标签必须闭合，属性必须有值
  • whitespaceTextNode 默认true， false表示如果某个节点是空白字符串，parse 时会删除
  • EnforceEncoding 默认 null, 会自动在传入的 html 代码里寻找
  • CleanupInput 默认 false, 如果为 true 则跳过解析器的整个清理阶段（后面三个也被忽略）
  • removeScripts 默认 true
  • RemoveStyles 默认 true
  • preserveLineBreaks 默认 false, 如果设置为 true 则会保留换行符

Modify the Dom

使用 setAttribute 修改节点的属性

$dom = new Dom;
$dom->load('<div class="all"><p>Hey bro, <a href="google.com">click here</a><br /> :)</p></div>');
$a = $dom->find('a')[0];
$a->setAttribute('class', 'foo');
echo $a->getAttribute('class'); // "foo"

//或者先 getTag 获得 tag 成员，然后调用 tag 的 setAttribute 方法
$tag = $a->getTag();
$tag->setAttribute('class', 'foo');
echo $a->getAttribute('class'); // "foo"

删除元素

$a->delete();
unset($a);
echo $dom; // '<div class="all"><p>Hey bro, <br /> :)</p></div>');
